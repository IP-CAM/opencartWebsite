1. Getting Started

Firstly, we would like to thank you for downloading our template, Free LaddyLike of opencart version 2.3.0.2. This manual consists of several parts which show the entire process of setting up and managing our template from scratch. We did our best to make this manual as clear and easy to follow as possible.

If you have any queries regarding installation or concerns about our template which you are not able to find in the documentation, feel free to ask for help from our support team, all you have to do is submit a ticket via our Support Ticket system.

2. Extract and Installation

2.1 Extract packages

Once download is completed, Extract or unzip the file and you could see the following packages:

  • Guides - our detailed documentation for the theme
  • Theme folder - for manual installation
  • Source - the PSD files of the theme
  • Licensing - the theme license

2.2 Theme Installation

Before you install this theme:

  1. Read our complete instruction to install theme
  2. Make sure that you have installed an OpenCart and running the correct version of it i.e., OpenCart 2.3.0.2
  3. Take a FULL BACKUP for your site

We recommend you to use our template on a newly installed OpenCart i.e., without custom core modifications in it.

Steps to install Free LaddyLike Theme :

Step 1 - Unzip the theme package

After unzipping downloaded package, please unzip the theme package - FreeLaddylike_OPC2.3.0.2.zip

Step 2 - Upload theme to your OpenCart site root directory

Note : If you want to enable languages then you can download it from this site http://www.opencart.com .
Download language from given url and paste it to your language directory.
Language directory root:"upload/catalog/language".

3. Theme Configuration

3.1 Activate Theme

Once the theme is uploaded, open your admin panel and navigate to Admin Panel > Extensions > Extensions so you could see the following screen.So now you are on the page named "Extensions".

Following are the steps to edit your store theme.

Step 1: Select and set "Themes(1)" from the filter section.

Step 2: Next, hit Edit.

Step 3: Next, select "laddyfree" from theme directory in general section.

Step 4: Set Status to 'Enabled'.

Step 5: Save.

3.2 Upload Logo

To change Store logo,

Step 1: Go to System > Settings > Edit > Image ,

Step 2: Click the Store logo file and set your Store logo.

Step 3: Click on “Save” button. 

3.3 Image Size Adjustment

Step 1: Go to Extensions > Extensions > Themes > Edit ,

Step 2: Change Category Image Size to: 904 X 235,

Step 3: Change Product Image Thumb Size to: 348 X 444,

Step 4: Change Product Image Popup Size to: 870 X 1110,

Step 5: Change Product Image List Size to: 274 X 350,

Step 6: Change Additional Product Image to: 88 X 112,

Step 7: Change Related Product Image size to: 274 X 350,

Step 8: Click on “Save” button.

4. Banner Settings

To add new banners, go to Admin > Design > Banners from your site admin panel.Now you could see the following screen named Banners containing all banner list.

Step 1: Click on edit button of HP Products and disable it.

Step 2: Save.

Step 3: To add a new banner, click on “Insert” button.

Or

Step 3: To edit a banner,click on “Edit” button.

4.1 Add Home Page Slideshow Banner

To Add Main Slideshow Banner, please follow the last step i.e.,Step number 3 from the above listed steps for "Edit".

Following are the steps to configure Main Slideshow Banner:

Step 1: Add/Edit Banner Name field.

Step 2: In Status field,set it to Enabled.

Step 3: Give the title of your banner and click on image to upload your banner.

Step 4: Again click on “Add Banner” button and upload your rest of two banners.

Step 5: Click on “Save” button.

4.2 Add LeftBanner

Follow the same steps listed in 4.1,to Add Left Banner to your store.

Note : LeftBanner contains only 1 banner, so you have to add only one banner in LeftBanner.

4.3 Add Productpage Banner

Follow the same steps listed in 4.1,to Add Productpage Slider to your store.

Note : Productpage Banner contains only 1 banner, so you have to add only one banner in Productpage Banner.

Get Support

If you have any queries regarding installation or concerns about our template which you are not able to find in the documentation, please go to our forum and post up a new topic with all the details we need. Please be sure to include your site URL as well or send us an E-Mail with the details of cpanel & FTPs .

If you have any query regarding to default OpenCart System & its configuration then please refer OpenCart site.

Our support scope

Our support covers configuration, building site as demo, trouble using any features, and bug fixes related to our template.In our "Free Templates" Support will be limited and will depend upon the issue, Rather then this in "Free Template" there will be no exta support. We don't provide support for customizations or 3rd party extension.

Our support time

We try our best to monitor the email around the clock, however, this is not always possible due to different time zones. We will try to reply you as fast as we can.

Support requests are being processed on business days from 9:30AM to 7:00PM IST from Monday to Saturday, [normally] within 24h in the order they were received, in case of Sunday or official holidays it can take more time. So if ask for help in the weekend, you may have to wait a little bit for support and please be patient.
